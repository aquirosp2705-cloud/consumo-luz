Lógica replicada y simplificada del Excel El consumo mensual se calcula con la lectura: Lectura actual - Lectura anterior. Con ese consumo se calculan los valores unitarios: Consumo $/kWh, Impuestos $/kWh, Total $/kWh y Subsidio $/kWh. El porcentaje inyectado se calcula como Inyectado kWh / (Consumo mes kWh + Inyectado kWh). No se muestran acumulados porque el objetivo es analizar el comportamiento de cada factura mensual.
